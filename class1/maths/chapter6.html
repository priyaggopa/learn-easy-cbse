A pattern is something that repeats again and again. Patterns are all around us! We can see patterns in numbers, colors, shapes, and even in nature. Learning patterns helps us understand the world better and makes math fun!

üî¢ Types of Patterns

1. Number Patterns

Number patterns are numbers that follow a rule and repeat.

Example 1: Counting by 1s

1, 2, 3, 4, 5, 6, 7, 8...

Example 2: Counting by 2s

2, 4, 6, 8, 10...

Example 3: Repeating pattern

1, 2, 1, 2, 1, 2...

2. Color Patterns

Color patterns use different colors that repeat in order.

Example 1: Red, Blue, Red, Blue...

Example 2: Yellow, Green, Purple, Yellow, Green, Purple...

3. Shape Patterns

Shape patterns use different shapes that repeat in order.

Example 1: Circle, Square, Circle, Square...

Example 2: Triangle, Circle, Square, Triangle, Circle, Square...
